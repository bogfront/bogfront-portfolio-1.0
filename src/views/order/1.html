Набросок страницы

Сайт будет адаптивным. Верхнее меню будет приклеено сверху страницы. На главный банер будет установлена привлекательная фотография, продающий текст и форма обратной связи. Инфографика будет показывать приемущества: иконка + текст. Карточка товаров будет иметь название, описание услуги, фотографии в виде слайдов и стоимость. Форма обратной связи для заказа звонка. Описание компании и сотрудников. На контакта будет вся информация, по которой можно связаться и карта(по желанию).
